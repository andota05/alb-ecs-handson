# alb-ecs-handson

ALB-ECS構築ハンズオン

## 留意事項

- ALB～ECSで構築、疎通の確認をする
- アプリケーションは作りこまない
- nginxのdefaultページが表示できればOKとする
- ALBのポートは80(HTTP)のみ使用
- VPCはdefault前提

## ハンズオン

- ALB、セキュリティグループ、ECS構築など詳細な手順は割愛

## 構成要素

- ALB × 1
  - Target-Group × 1
- Security Group × 2
  - for-ALB
  - for-ECS
- ECS × 1
  - cluster
  - task

### 手動構築時の気づき

#### ALB

- 段階踏むためには、ALBのデフォルトアクションを固定レスポンスにして何かメッセージが表示、レスポンスされるかという確認を挟むとよい
- ターゲットグループはALBが「次どこに繋げるかの道しるべ」(ECSの居場所を書置きしてあげる)
- ターゲットグループを先に作成してよいが、ターゲットはECSなどインスタンスが出来上がってからあとから登録が必要

#### ECS

- クラスター、タスク定義、サービス、SGなど設定事項が多々あって大変
- タスク定義のコンテナ-イメージURIに「nginx」と指定すればイメージ参照できるよ
- ecsに設定するセキュリティグループのソースにalbに設定するセキュリティグループを設定することで「albから来た接続のみを通すようにできる」★POINT★
- ecsのタスクを定義してから、そのプライベートIPをALBのターゲットグループ-ターゲットに追加する
- 5回ぐらい手動で構築すれば流れとイメージがつく
